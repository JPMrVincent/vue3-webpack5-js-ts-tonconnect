# vue3-webpack5-js-ts-tonconnect

Vue code demo based on ton connect, include connect wallet, get signature, construct Ton Network USDT transaction, a complete set of Ton SDK usage.

Demo project by vue3, pure js writing method, supports setup, supports ts parsing, uses webpack5 for packaging, supports multiple environments and multiple variables.
